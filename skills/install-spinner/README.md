# Install Spinner

Install themed spinner verb packs for Claude Code directly from GitHub.

## Installation

```bash
npx skills add alexpl292/awesome-claude-spinners
```

## Usage

Once installed, ask your agent:

```
Install a spinner pack
```

You'll be presented with 35+ themed packs to choose from, including:

- **Developer** - Programming & dev culture
- **Pirate** - Pirate speak
- **Corporate** - Corporate buzzwords & jargon
- **Space** - Space & sci-fi
- **Wizard** - Fantasy & magic themed
- **Vim** - Vim editor enthusiasts
- And many more!

The selected pack will be installed into your `~/.claude/settings.json`.
